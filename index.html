☁️ Cloud Sync — ใช้งานหลายเครื่อง
วิธีตั้งค่า Firebase (ฟรี ใช้เวลา ~5 นาที)
  1. ไปที่ console.firebase.google.com → เข้าสู่ระบบด้วย Google
  2. กด Add project → ตั้งชื่อ (เช่น "saffron-dashboard") → Create
  3. ในเมนูซ้าย → Build → Realtime Database → Create database
  4. เลือก location ใกล้ที่สุด → เริ่มใน test mode → Enable
  5. กลับหน้าแรก Project → ⚙️ Project settings → Your apps → เพิ่ม Web app (</>)
  6. ตั้งชื่อ app → ก็อปปี้ firebaseConfig object ทั้งหมด → วางในช่องด้านล่าง
